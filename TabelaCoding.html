Linguagens de Programação

Linguagem Aplicação Opinião
Javascript Desenvolvimento web Ótima para criar interações em páginas web.
Java Desenvolvimento de aplicativos Uma linguagem robusta para todo tipo de desenvolvimento que está a 20 anos em alta.
PHP Desenvolvimento web backend Boa para criação de sites, mas tem algumas limitações.
Python Aplicações diversas Uma linguagem versátil e de fácil aprendizado.
C# Desenvolvimento de software Windows Escolha para desenvolvimento de aplicativos Windows.
Kotlin Desenvolvimento de aplicativos Uma ótima linguagem para desenvolvimento, utilizando do melhor do java e aprimorando.
